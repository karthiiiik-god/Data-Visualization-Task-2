# Data Visualization & Storytelling - Task 2

## Objective
To analyze sales data and create meaningful visualizations that support business decision-making.

## Tools Used
- Power BI Desktop
- Superstore Sample Dataset

## Key Insights
1. Technology category generates highest revenue.
2. Some regions have strong sales but lower profit margins.
3. High discounts negatively impact profitability.
4. Sales show consistent growth over time.

## Conclusion
The company should optimize discount strategy and focus on high-profit categories to improve margins.
